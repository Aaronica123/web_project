### Project Overview
**JumShop** is a complete, multi-page e-commerce website built from scratch using **only HTML5, CSS3, and Vanilla JavaScript** – no frameworks, no libraries.

It replicates the core experience of **Jumia** (Africa’s leading online marketplace) and includes:

- Responsive, mobile-first design
- Persistent shopping cart (`localStorage`)
- Live product search & filtering
- Image zoom on product detail page
- Form validation with instant feedback
- Smooth animations & hover effects
- Fixed navigation + mobile hamburger menu
- Clean, production-ready folder structure

---

### Website Purpose
To provide a fast, intuitive shopping experience where users can:
1. Browse featured and all products
2. Search and filter items instantly
3. View detailed product pages
4. Add/remove items from a persistent cart
5. Contact support via validated form

### Pages & User Flow

| Page              | File                  | URL                     | Key Features |
|-------------------|-----------------------|-------------------------|--------------|
| **Home**          | `index.html`          | `/index.html`           | Hero banner, featured deals, quick add-to-cart |
| **Products**      | `products.html`       | `/products.html`        | Live search, grid layout, view details |
| **Product Detail**| `product-detail.html` | `/product-detail.html`  | Large image with zoom, detailed info |
| **Cart**          | `cart.html`           | `/cart.html`            | Dynamic cart, remove items, total price |

**Typical User Journey:**

Home → Products → Product Detail → Add to Cart → Cart → Contact (optional)

### Features Implemented
- **Responsive Design** – Perfect on phones, tablets, desktops
- **Mobile Navigation** – Hamburger menu with slide-down animation
- **Persistent Cart** – Items stay after page refresh (`localStorage`)
- **Live Search** – Instant filtering as you type
- **Image Zoom** – Click product image → 1.5× zoom
- **Remove Items** – Delete individual products from cart
- **Clear Cart** – One-click empty cart
- **Form Validation** – Real-time checks + success message
- **Smooth Animations** – Card lift, button pulse, page transitions
- **Semantic HTML5** – `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`
- **No External Dependencies** – Pure vanilla code
